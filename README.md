## WHEATER-CLI-KG

To get weather in realtime in kyrgyz language.

**1. Install**

```shell
npm i weather-cli-kg
```

**2. ## COMMANDS**

|             |             |
| ----------- | ----------- |
| [`weather`] | Get weather |
| [`-t`]      | Set token   |
| [`-s`]      | Set city    |
| [`-h`]      | Get help    |
